## About SDC INDIA

**Student Developers Community (SDC INDIA)** is a student-driven developer community focused on helping students gain practical skills in software development, artificial intelligence, and emerging technologies.

We organize:

-  Hackathons
-  AI Workshops
-  Developer Bootcamps
-  Tech Events
-  Networking Sessions

## 📚 Project Documentation

To help the team get started, we have provided the following resources in the `docs/` folder:

- [**Style Guide**](docs/style_guide.md): Color palette, typography, and design consistency rules (including Aceternity UI recommendation).
- [**Task List**](docs/task.md): A detailed checklist for initial setup and feature implementation.
- [**Walkthrough**](docs/walkthrough.md): A summary of the current base stack and how to start development.

## 🚀 Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
3.  **Open in Browser**: [http://localhost:3000](http://localhost:3000)
